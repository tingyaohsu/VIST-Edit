README for VIST-Edit Dataset
============================

This is the README file for the supplementary materials of submission #384, entitled
"Visual Story Post-Editing," to ACL 2019 short paper.

=== JSON Attributes ===

[auto_story_text_normalized]
The automatic story generated by the model.

[edited_story_text]
The original text that writers submitted.

[normalized_edited_story_text_sent]
The [edited_story_text] normalized by NLP tools.

=== Note ===

This data and readme file are for reviewers and AC/PC of ACL 2019 only. Please do
not redistribute without our permission. Thanks.
